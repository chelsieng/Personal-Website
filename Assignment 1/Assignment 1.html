Assignment 1: Web Site Description

Purpose of Web Site

My web site will be a personal one. It will be used as my online portfolio where I will be displaying information about my professional life as well as hobbies and interests.

General Description

The main aim of my web site is to promote myself, build and control my personal brand. On the professional level, employers will find me more easily and will have more convenience to access my projects. I believe having a personal web site is more dynamic than a simple resume. I want to present who I am my own way and express my real personality. The major sections of the web site is composed of:

  • Learn more about me which covers an introduction about me.
  • About which covers the incentive and goal of my web site.
  • Projects which displays source codes and links to GitHub repositories.
  • Q&A section which is a discussion board among the users and myself.
  • Media which features any significant photos or videos.
  • Contact me which allows the user to get in touch with me.
  • Follow me which allows the user to be kept posted about any updates about me.
